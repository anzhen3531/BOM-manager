# `Windchill Secondary Development`



## 概述

项目为一个简单的`Windchill`二次开发项目，用于学习`Windchill` 基本`API`、及开发一些公用的工具



依赖环境：

`JDK`: 1.8

`Windchill Version` :  11.M030



:red_circle:注意事项

> 此项目代码需要在`Windchill`中执行，如果没有`Windchill`，这项目没有任何作用
